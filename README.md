# Capistrano::SSH

SSH tasks for Capistrano v3.*

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'capistrano', '~> 3.0.0'
gem 'capistrano-ssh'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install capistrano-ssh

## Usage

Require the module in your `Capfile`:

```ruby
require 'capistrano/ssh'
```
